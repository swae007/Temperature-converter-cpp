# Temperature Converter

A C++ converter that takes any temperature scale 
and outputs all other scales simultaneously.

## Features
- Converts between Celsius, Fahrenheit and Kelvin
- Case-insensitive input handling
- Clean formatted output for all three scales

## How to run
1. Compile: g++ temperature.cpp -o temp
2. Run: ./temp

## What I learned
Conditionals, functions, user input handling 
and case-insensitive string comparison.
